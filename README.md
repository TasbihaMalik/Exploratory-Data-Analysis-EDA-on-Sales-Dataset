# 📖 Project Overview

This project demonstrates a complete **Exploratory Data Analysis (EDA)** workflow on a sales dataset using Python.

The objective is to understand the data, identify meaningful patterns, calculate statistical summaries, detect outliers, visualize relationships between variables, and generate business insights that support data-driven decision making.

The project follows professional data analysis practices and prepares the dataset for Business Intelligence tools such as **Power BI**.

---

# 🎯 Project Objectives

The objectives of this project include:

- Import and explore the dataset
- Understand dataset structure
- Perform descriptive statistical analysis
- Detect missing values
- Analyze numerical distributions
- Perform correlation analysis
- Detect outliers
- Analyze customer behavior
- Analyze product performance
- Analyze payment methods
- Analyze monthly sales trends
- Generate business insights
- Export reports for future dashboard development

# ▶️ Installation Guide

## Clone Repository

```bash
git clone https://github.com/TasbihaMalik/Exploratory-Data-Analysis-EDA-on-Sales-Dataset.git
```

---

## Navigate into the Project

```bash
cd Exploratory-Data-Analysis-EDA-on-Sales-Dataset
```

---

## Create Virtual Environment

```bash
python -m venv virtual
```

---

## Activate Virtual Environment

### PowerShell

```powershell
.\virtual\Scripts\Activate.ps1
```

### Git Bash

```bash
source virtual/Scripts/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Launch Jupyter Notebook

```bash
jupyter notebook
```

Open:

```
notebook.ipynb
```

Run all cells.
